jQuery width & height Methods

Methods are:

  • width() :- gives content width
  • height()
  • innerWidth() :- gives width till border start
  • innerHeight()
  • outerWidth() :- gives width till border end
  • outerHeight()
  • outerWidth(true) :- gives width till margin end
  • outerHeight(true)
